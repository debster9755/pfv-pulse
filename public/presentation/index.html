Vercel (Serverless)
Why Serverless?
Neon + Upstash scale to zero — no idle compute cost. Vercel handles all scaling automatically.
Why Playwright Isolated?
~300MB binaries exceed Vercel's 50MB limit. Isolated to Railway / Fly.io as a separate service.
Why No LLM for Routing?
Regex matching = 0ms + $0 + 100% reliable for structured pricing queries. LLM adds 500ms+.
Resilience

Graceful Degradation

Every external dependency has an explicit fallback. The app never throws a 500 error due to a missing key or failed service.

API Key Missing
isConfigured() returns false → returns [] → chat shows EmptyState
Redis Down
connect() throws → cacheGet returns null → API calls proceed uncached
Postgres Down
Prisma throws → try/catch returns empty array → UI shows EmptyState
Scraper Down
25s timeout → 503 with null data → UI shows "Data Unavailable" — no crash
Scraper Blocked
Playwright throws → returns partial results collected before failure
Module Disabled
enabled=false prop → EmptyState "Integration Disabled" shown in panel
0 Data Sources
Recommendation Engine → Action=HOLD · Confidence=LOW · Explanatory note shown — engine always outputs
